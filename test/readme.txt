Для работы тестов необходимо:
1. Установить библиотеки selenium== 4.9.0 pytest-selenium== 4.0.0 pytest== 6.2.5
2. Установить, если это необходимо, браузер Chrome.
3. Скачать с сайта https://googlechromelabs.github.io/chrome-for-testing/ драйвер версии вашего Chrome и поместить его в папку chromedriver
3. В Data необходимо заполнить переменные данными зарегистрированного пользователя

Тесты проверяют авторизацию на сайтах
